#### 获取token

#### 编辑配置文件

在指定位置输入token

~/.picgo/config.json

```
{
	"picBed":{
		"uploader":"smms",
		"smms":{
			"token":"your token"
		}
	},
	"picgoPlugins":{}
}
```
